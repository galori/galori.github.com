However you can still use the same pattern - when firing a global event, you can bind it to any element on the dom - so you can bind to $('body') or $(document): [sourcecode language="jscript"] $(document).bind("sneeze",function(){}) $.event.trigger("sneeze"); [/sourcecode] This is a great way to decouple code and when used like a messaging system, can really clean up your code.
